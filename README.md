### Overview

## Setup

To install the project run the command:

```
yarn install
```

### start app

To run the project run the command:

```
yarn start
```

### build app

To build the app run the command:

```
yarn build
```

### deploy app

To deploy the project run the command:

```
yarn deploy
```
